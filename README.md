# etl-pipeline-python
A simple etl pipeline built with python to process CSV data and load  it into a realational database.

## Description
The project extracts data from CSV files, performs basic data transformations,
and loads the processed data into a relational database.

## ETL Process
- Extract data from CSV files
- Transform data by cleaning and validating values
- Load data into a SQLite database

## Tools & Technologies
- Python
- Pandas
- SQLite
- SQL

## Goal
To demonstrate core data engineering concepts such as ETL pipelines,
data transformation, and structured data storage.
